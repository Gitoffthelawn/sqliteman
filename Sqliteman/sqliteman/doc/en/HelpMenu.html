The Help Menu

Help-> Help Content...

Open help window with sqliteman documentation. The help documentation is currently only available in English.

Help-> About...

Show build information for sqliteman and version details for sqlite, Qt, and the operating system, in the Error Log area.

Help->About Qt...

Dialog with Qt library version and credits.
